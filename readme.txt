Easily add and approve guest posts to your Wordpress blog with the official Wordpress Guest Post plugin. 

== Description ==

The Wordpress Guest Post plugin is a simple plugin that allows you to accept, edit, & publish guest posts to your blog without writers having to login to your Wordpress Admin area.

1. Easily obtain guest posts without user registration.
2. Users can add their posts right from a form hosted on a page on your blog. No logging in your Wordpress admin.
3. Effortlessly manage and edit post submissions
4. Customizable post submission notification on site and via email
5. Check out more information at: http://WordpressGuestPost.com

== Installation ==

1. Upload the unzipped Wordpress Guest Post files to the plugins directory
2. Activate the plugin through the 'Plugins' menu in Wordpress
3. Go to the settings page to select your guest post page and customize your settings

Get detailed installation instructions at: http://LisaAngelettieBlog.com/wordpress-guest-post-plugin/
